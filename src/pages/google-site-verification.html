Google Site Verification

This page is used for Google Search Console verification.

Please add your Google verification meta tag to the main layout file.

How to add Google verification:

  1. Go to Google Search Console
  2. Add your property: https://aichatwindow.com
  3. Choose "HTML tag" verification method
  4. Copy the meta tag provided by Google
  5. Add it to the <head> section of src/layouts/Layout.astro

Example:

<meta name="google-site-verification" content="YOUR_VERIFICATION_CODE" />
